# Bundled web fonts

The site bundles the Latin web-font files used by its two type families so
production builds do not depend on a third-party font service.

- Atkinson Hyperlegible, regular and bold. Copyright 2020 Braille Institute
  of America, Inc.
- Barlow Condensed, bold. Copyright 2017 The Barlow Project Authors
  (<https://github.com/jpt/barlow>).

Both families are distributed under the SIL Open Font License, Version 1.1.
The license text is included in `OFL.txt`. The files are unmodified Google
Fonts Latin-subset WOFF2 distributions.
